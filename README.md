# Resolution-Game_Dev-Week-3

This game was made using Godot for [Resolution Game Dev](https://resolution.hackclub.com/app/pathway/game_dev), a [Hack Club](https://hackclub.com/) YSWS. It’s fully open source, and you’re free to use it however you like.

It’s a super simple platformer with only one level, a menu screen, and a complete screen. It’s not a full game, but more of a experince.
